Pendahuluan

Seven segment display adalah komponen elektronika yang sangat populer digunakan untuk menampilkan angka dan beberapa huruf. Dalam dunia elektronika digital, terdapat dua jenis seven segment display yang paling umum: anoda bersama (common anode) dan katoda bersama (common cathode).

Kedua jenis ini memiliki karakteristik berbeda yang penting untuk dipahami sebelum menggunakannya dalam rangkaian elektronika. Sebagai tim yang terdiri dari dua orang, kami akan menjelaskan perbedaan mendasar antara kedua jenis seven segment ini dan bagaimana cara menggunakannya dalam proyek elektronika.

Perbedaan Dasar

Seven Segment Common Anoda

Pada common anode, semua anoda dari LED dihubungkan menjadi satu pin bersama yang harus dihubungkan ke sumber tegangan positif (VCC). Untuk menyalakan segmen tertentu, kita harus memberikan logika LOW (0V) ke pin segmen tersebut melalui resistor pembatas arus.

Pin Bersama → VCC (+5V)
Segmen → LOW (0V) untuk menyala

Seven Segment Common Katoda

Pada common cathode, semua katoda dari LED dihubungkan menjadi satu pin bersama yang harus dihubungkan ke ground (0V). Untuk menyalakan segmen tertentu, kita harus memberikan logika HIGH (+5V) ke pin segmen tersebut melalui resistor pembatasi arus.

Pin Bersama → GND (0V)
Segmen → HIGH (+5V) untuk menyala

Cara Mengidentifikasi Jenis Seven Segment

Sebagai tim, kami menemukan beberapa metode untuk mengidentifikasi jenis seven segment:

1. Dengan Multimeter

  • Atur multimeter pada mode dioda
  • Hubungkan probe merah ke pin bersama, hitam ke pin segmen
  • Jika segmen menyala → Common anode
  • Jika tidak, coba sebaliknya

2. Dari Datasheet

  • Selalu periksa datasheet dari produsen
  • Biasanya terdapat diagram pin yang jelas

3. Pengujian Sederhana

  • Hubungkan pin bersama ke VCC melalui resistor 220Ω
  • Sentuh pin segmen ke GND
  • Jika menyala → Common anode
  • Jika tidak, coba kebalikannya

Contoh Implementasi dengan Arduino

Berikut adalah contoh kode untuk menampilkan angka 5 pada kedua jenis seven segment:

Contoh Kode Arduino untuk Seven Segment
// Deklarasi pin untuk Common Anode
int pinCA[] = {2,3,4,5,6,7,8}; // a,b,c,d,e,f,g
int commonAnodePin = 9;

// Deklarasi pin untuk Common Cathode  
int pinCC[] = {10,11,12,13,A0,A1,A2}; // a,b,c,d,e,f,g
int commonCathodePin = A3;

// Pola untuk angka 5 (segmen a,f,g,c,d)
byte angka5 = B01101101;

void setup() {
  // Inisialisasi pin Common Anode
  for(int i=0; i<7; i++) {
    pinMode(pinCA[i], OUTPUT);
  }
  pinMode(commonAnodePin, OUTPUT);
  digitalWrite(commonAnodePin, HIGH); // Common anode ke VCC
  
  // Inisialisasi pin Common Cathode
  for(int i=0; i<7; i++) {
    pinMode(pinCC[i], OUTPUT);
  }
  pinMode(commonCathodePin, OUTPUT);
  digitalWrite(commonCathodePin, LOW); // Common cathode ke GND
}

void loop() {
  // Tampilkan angka 5 pada Common Anode
  tampilkanAngka(pinCA, angka5, true);
  
  // Tampilkan angka 5 pada Common Cathode
  tampilkanAngka(pinCC, angka5, false);
  
  delay(2000);
}

void tampilkanAngka(int pins[], byte pola, bool isAnoda) {
  for(int i=0; i<7; i++) {
    // Untuk anoda: bit 1 = segmen MATI (HIGH), bit 0 = segmen NYALA (LOW)
    // Untuk katoda: bit 1 = segmen NYALA (HIGH), bit 0 = segmen MATI (LOW)
    bool segmenMenyala = bitRead(pola, i);
    
    if(isAnoda) {
      digitalWrite(pins[i], segmenMenyala ? LOW : HIGH);
    } else {
      digitalWrite(pins[i], segmenMenyala ? HIGH : LOW);
    }
  }
}

Kelebihan dan Kekurangan

Common Anode

  • Kompatibel dengan sink-current dari mikrokontroler
  • Lebih umum tersedia di pasaran
  • Biasanya lebih murah
  • Cocok untuk aplikasi dengan banyak display

Common Anode

  • Logika terbalik (LOW untuk menyalakan)
  • Kurang intuitif untuk pemula
  • Membutuhkan driver transistor untuk arus tinggi

Common Cathode

  • Logika lebih intuitif (HIGH untuk menyalakan)
  • Cocok untuk driver sumber-arus (source-current)
  • Mudah dipahami oleh pemula
  • Kompatibel dengan banyak IC driver

Common Cathode

  • Kurang umum dibanding common anode
  • Terkadang lebih mahal
  • Membutuhkan perhatian ekstra pada konfigurasi pin

Kesimpulan

Pemahaman tentang perbedaan antara seven segment common anode dan common cathode sangat penting dalam merancang rangkaian elektronika. Kedua jenis memiliki aplikasinya masing-masing tergantung pada kebutuhan dan konfigurasi rangkaian.

Sebagai tim, kami merekomendasikan untuk selalu memeriksa datasheet sebelum menggunakan seven segment display untuk menghindari kesalahan dalam penyambungan. Dengan memahami konsep dasar ini, Anda dapat dengan mudah mengintegrasikan seven segment display ke dalam proyek elektronika Anda, baik untuk keperluan edukasi maupun aplikasi praktis.
